# 贪心

## 1. 核心思想
每一步选择当前最优方案，并保证该选择不会破坏后续全局最优。

## 2. 适用题型
- 区间选择、任务调度
- 最少操作、最多满足
- 跳跃覆盖、资源分配
- 排序后按规则选择

## 3. 经典例题和逐步拆解
LeetCode 455：分发饼干。

1. 孩子胃口和饼干尺寸分别升序排序。
2. 用最小的、能够满足当前孩子的饼干完成匹配。
3. 饼干太小时只移动饼干指针。
4. 成功匹配后两个指针都移动。

复杂度主要来自排序：`O(n log n + m log m)`。

## 4. 易错点
- 忘记排序，使局部选择失去方向。
- 饼干太小时错误跳过孩子。
- 匹配成功后只移动一个指针。
- 只凭感觉贪，无法解释为什么局部最优安全。

## 6. 精华总结
做贪心题要回答两句：每一步贪什么？为什么这样不会让未来更差？
