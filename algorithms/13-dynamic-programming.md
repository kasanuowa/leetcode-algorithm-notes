# 动态规划入门

## 1. 核心思想
当前答案由更小规模问题的答案推出。写代码前明确：状态含义、转移方程、初始化。

## 2. 适用题型
- 方法数
- 最大/最小收益
- 选或不选
- 能否凑出目标
- 序列最优问题

## 3. 经典例题和逐步拆解
LeetCode 70：爬楼梯。

定义 `dp[i]` 为到达第 `i` 阶的方法数。最后一步只能从 `i-1` 或 `i-2` 到达：

```js
dp[i] = dp[i - 1] + dp[i - 2];
```

初始化 `dp[1] = 1`、`dp[2] = 2`。复杂度：时间 `O(n)`，空间可优化到 `O(1)`。

## 4. 易错点
- 没先定义 `dp[i]` 的准确含义。
- 初始化与循环起点不匹配。
- 混淆“方法数的求和”和“最优值的 max/min”。
- 状态转移中使用了尚未计算的状态。

## 5. 今日练习题
LeetCode 198：打家劫舍。

```js
dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i]);
```

## 6. 精华总结
DP 三问：`dp[i]` 是什么、从哪里来、最小问题答案是什么。
