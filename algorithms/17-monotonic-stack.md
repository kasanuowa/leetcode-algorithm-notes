# 单调栈

## 1. 核心思想
栈中保存尚未找到答案的候选元素，并维持单调关系。当前元素打破单调性时，为被弹出的元素确定答案。

```js
const stack = [];
for (let i = 0; i < nums.length; i++) {
  while (stack.length && nums[i] > nums[stack[stack.length - 1]]) {
    const index = stack.pop();
    // nums[i] 是 nums[index] 右侧第一个更大元素
  }
  stack.push(i);
}
```

## 2. 适用题型
- 左/右侧第一个更大元素
- 左/右侧第一个更小元素
- 每日温度、下一个更大元素
- 柱状图最大矩形、接雨水

## 3. 经典例题和逐步拆解
LeetCode 739：每日温度。

1. 栈中保存仍在等待更高温度的日期下标。
2. 当前温度高于栈顶日期时，弹栈。
3. 等待天数为 `i - previousIndex`。
4. 使用 `while`，因为当前温度可能解决多个日期。

复杂度：时间 `O(n)`，空间 `O(n)`；每个下标最多入栈、出栈一次。

## 4. 易错点
- 需要计算距离时却只存值。
- 使用 `if` 只解决一个旧元素。
- “更高”题目错误使用 `>=`。
- 先把当前元素入栈再比较。
