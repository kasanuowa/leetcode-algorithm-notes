# 网格 DFS

## 1. 核心思想
从一个格子出发，沿上下左右访问整个连通区域。固定流程：判断、标记、扩散。

```js
function dfs(r, c) {
  if (r < 0 || r >= rows || c < 0 || c >= cols || 不符合条件) return;
  标记当前格子;
  dfs(r + 1, c);
  dfs(r - 1, c);
  dfs(r, c + 1);
  dfs(r, c - 1);
}
```

## 2. 适用题型
- 岛屿数量、最大岛屿面积
- 洪水填充、连通块染色
- 二维矩阵中的区域搜索

## 3. 经典例题和逐步拆解
LeetCode 200：岛屿数量。

1. 遍历所有格子。
2. 每遇到一个未访问陆地，岛屿数加一。
3. DFS 将整片相邻陆地标记为已访问。
4. 后续遍历不会重复计数。

复杂度：时间 `O(mn)`，递归空间最坏 `O(mn)`。

## 4. 易错点
- 没有立刻标记，导致相邻格子反复访问。
- 越界条件漏掉一边。
- 混淆数字 `1` 与字符串 `"1"`。
- 队列或栈里只放格子值，而不是坐标。

## 5. 今日练习题
LeetCode 733：图像渲染。若原颜色等于目标颜色，应提前返回。

## 6. 精华总结
二维连通区域优先想 DFS/BFS；只统计完整区域时，DFS 通常更直接。
